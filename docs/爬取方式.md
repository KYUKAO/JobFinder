# TA Job Finder 爬取方式记录

> 本文档记录 JobFinder 项目中各数据源的具体爬取方式、数据格式及注意事项。

## 1. 国内岗位（51Job + 游戏公司官网）

### 51Job API

- **端点**: `https://we.51job.com/api/job/search-pc`
- **方法**: GET + 移动端 API 绕过直接访问
- **参数**:
  - `jobArea` = 城市名
  - `keyword` = 搜索关键字（如 `技术美术 TA`、`TA 技术美术`）
  - `page` = 页码
  - `jobAreaCode` = 城市码
  - `compact` = `true`
- **返回**: JSON，内含 `resultbody.job[]` 数组
- **注意**: 需要设置 User-Agent，移动端优先
- **超时**: 30s，失败重试 3 次，backoff factor 2s

### 游戏公司官网

| 米哈游 | `https://jobs.mihoyo.com/#/positions?keyword=TA` |
| 字节跳动游戏 | `https://job.bytedance.com/technology` |

- **解析**: 检测 Content-Type，XML 用 `xml` 解析器，JSON 跳过，HTML 用 `lxml`
- **异常处理**: 单独捕获 SSLError / Timeout / ConnectionError / 404 / 其他异常

## 3. TA 作品集（B站）

### API 方式（优先）

- **端点**: `https://api.bilibili.com/x/web-interface/search/type`
- **参数**: `search_type=video`, `keyword=<关键字>`, `page=1`
- **返回**: JSON，`data.result[]` 数组

**可用关键字列表**（2026-04 测试通过）:
```
UE渲染, 渲染引擎, 游戏特效, UE特效, VFX, 粒子特效,
技术美术, 图形程序, TA教程, houdini教程, 程序化生成,
游戏美术, Unity shader, game vfx, unity教程, 场景美术, 动画特效
```

- **过滤**: 每个关键字取前 10 条结果，检测到 412 状态码则跳过
- **并发控制**: 每请求间隔 1-2s 随机
- **Header**: `User-Agent: Mozilla/5.0`, `Referer: https://www.bilibili.com/`

### 相关推荐备选（API 被封时）

- **端点**: `https://api.bilibili.com/x/web-interface/popular?pn=1&ps=20`
- 获取热门视频后，按关键字过滤出 TA 相关视频

### BV API 详情

- **端点**: `https://api.bilibili.com/x/web-interface/view?bvid=<BV号>`
- 获取视频详细信息：播放量、时长、标签等

### 作品集字段

```json
{
  "bvid": "BV1xx411c7XZ",
  "title": "UE渲染 | 赛博朋克场景",
  "author": "UP主名",
  "description": "视频描述...",
  "duration": "12:34",
  "views": 12345,
  "category": "TA",
  "platform": "bilibili",
  "url": "https://www.bilibili.com/video/BV1xx411c7XZ",
  "addedDate": "2026-04-18"
}
```

### B站 API 注意事项

- **412 被封**: 大批量请求时会触发风控，需要切换 User-Agent 或降速
- **referer**: 必须带 `https://www.bilibili.com/` 否则被拦截
- **Session**: 使用 requests.Session() 保持 Cookie

## 4. 数据保存格式

所有数据保存在项目根目录 JSON 文件中：

| 文件 | 内容 | 最大条数 |
|------|------|---------|
| `jobs-domestic.json` | `{"updated":"...", "domesticJobs":[...]}` | 400 |
| `jobs-overseas.json` | `{"updated":"...", "overseasJobs":[...]}` | 400 |
| `portfolios.json` | `{"updated":"...", "portfolios":[...]}` | 200 |
| `summary.json` | 市场分析摘要 | - |
| `github_contents.json` | 合并导出 | - |

## 5. 运行方式

```bash
# 完整爬取
python scripts/updater.py

# 只爬取国内岗位
SCRAPE_TARGET=domestic python scripts/updater.py

# 只爬取海外岗位
SCRAPE_TARGET=overseas python scripts/updater.py

# 只爬取作品集
python scripts/run_portfolios.py   # 或在 updater.py 中设置
```

GitHub Actions 每天北京时间 8:00 自动运行完整爬取。

## 6. 常见问题排查

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 51job 返回空 | API 被拦截 | 更换 User-Agent，增加 timeout 到 30s |
| 51job 返回 HTML | 触发了人机验证 | 切换到移动端 API 或改用 Boss 直聘 |
| 游戏公司 404 | URL 变更 | 更新 companies 列表中的 URL |
| B站 412 | 请求频率过高 | 降速到 2-3s 间隔，更换 User-Agent |
| B站 search API 空 | 关键字被封 | 切换到相关推荐备选方案 |
| 合并后字段丢失 | 数据格式不统一 | 加载时做兼容性处理 `get('domesticJobs', data if isinstance(data, list) else [])` |
